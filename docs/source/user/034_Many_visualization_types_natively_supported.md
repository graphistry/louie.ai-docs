# Many Visualization Types Natively Supported

Many visualization types natively supported for comprehensive data analysis and presentation.

## Supported Types

**Graphs**
- Graphistry

**Maps**
- kepler.gl

**Charts**
- Perspective
- w/ Plotly

**+ More:** Code agent can generate anything
