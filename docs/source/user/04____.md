# Talking to Your Data

What if we could talk to our data and get smart & interactive answers back?

Louie is a GenAI-first analyst platform for talking to data (with APIs!) that provides:

* GPU-accelerated visual graph AI platform
* Generative AI + Graph AI capabilities for databases, documents, and more

## Key Capabilities

###### STRUCTURE
Generative AI + Graph AI: Databases, documents, and more

###### TALK
GenAI-first analyst platform for talking to data (+ APIs!)

###### SEE
GPU-accelerated visual graph AI platform
