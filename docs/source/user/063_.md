# Reactive Bindings: Dashboard Mode

Dashboard mode provides an interactive interface for managing reactive bindings with dedicated input and output panels, enabling streamlined control over your notebook's reactive workflow.

## Key Features

- **Bound inputs**: Centralized input management panel
- **Bound outputs**: Dedicated output visualization area
- **Dependency tracking**: Visual representation of data relationships
- **Selective execution**: Rerun individual components or entire workflow
- **Interactive controls**: Real-time input modification and execution
